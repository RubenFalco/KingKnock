Aqui está um passo a passo detalhado para criar e hospedar seu site no GitHub Pages. Vamos lá!

Passo 1: Crie uma conta no GitHub

  1. Acesse GitHub e clique em Sign up para criar uma conta, caso ainda não tenha.
  2. Complete o cadastro seguindo as instruções.

Passo 2: Crie um repositório

  1. Após fazer login no GitHub, clique no botão + no canto superior direito e selecione New repository.
  2. Preencha os campos:
    • Repository name: Coloque um nome, como derruba-o-rei.
    • Deixe o repositório como Public.
    • Marque a opção Add a README file.
  3. Clique em Create repository.

Passo 3: Adicione o seu código

  1. No repositório recém-criado, clique no botão Add file e depois em Upload files.
  2. Arraste e solte o arquivo index.html que você criou para essa tela.
  3. Clique em Commit changes para salvar o arquivo no repositório.

Passo 4: Ative o GitHub Pages

  1. No seu repositório, clique na aba Settings (menu superior).
  2. Role a página para baixo até a seção Pages no menu lateral esquerdo.
  3. Em Source, selecione a branch main no menu suspenso e clique em Save.
  4. O GitHub irá gerar um link para o seu site (algo como https://seu-usuario.github.io/derruba-o-rei).

Passo 5: Acesse o site

  1. Aguarde alguns minutos enquanto o site é configurado.
  2. Clique no link gerado pelo GitHub Pages (mostrado na seção Pages).
  3. Seu site estará online!

Dica Extra: Atualizando o site

  • Para fazer alterações no seu site, basta voltar ao repositório, editar o código no arquivo index.html, e clicar em Commit changes para salvar.
